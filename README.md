# CPDEV-Site

Diese Distribution wird von der [conPassione gmbh](https://www.conpassione.ch)
zur Verfügung gestellt.

Diese Distribution ist gedacht als Entwicklungsumgebung für eigene
Extensions. Sie kann als Dependency im Abschnitt require-dev in der composer.
json der Extension
aufgeführt werden.

Die Verfügbarkeit für TYPO3-Versionen ist in der composer.json aufgeführt.
